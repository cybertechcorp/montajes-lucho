# React Template

This project is based on the following React template repository https://github.com/javierhersan/react-template, designed to accelerate web development. The primary goal is to enable developers to bring their ideas to the market as quickly as possible, with an integrated go to market strategy.

## Installation

This is the installation guide.

1. First, install NodeJS v22 or greater.
2. Second install a package manager (npm, pnpm or yarn)
3. Clone the project in your local machine

```console
$ git clone https://github.com/cybertechcorp/montajes-lucho.git
```

4. Install all the dependencies of the project.

```console
$ pnpm install
```

5. Run the project

```console
$ pnpm start
```
